# AI-Helper
AI 쉽게 접속할 수 있도록 돕는 Electron 기반 프로그램
- 싱글모드 : chatgpt 1개 지원
- 멀티모드 : chatgpt, perplexity AI 2개 지원

## 2. 주요 기능
- 축소, 확대, 원복 단축키 기능 추가 (Command or Ctrl + -, Command or Ctrl + =, Command or Ctrl + 0)
- 싱글 모드(Command or Ctrl + 1), 멀티 모드 추가 (Command or Ctrl + 2)
